# SportzHub Images Directory

This directory should contain the following image files:

## Required Images:

1. **logo.png** - SportzHub logo (recommended size: 200x60px)
2. **futsal.jpg** - Futsal court image (recommended size: 800x600px)
3. **badminton.jpg** - Badminton court image (recommended size: 800x600px)  
4. **padel.jpg** - Padel court image (recommended size: 800x600px)
5. **default-court.jpg** - Default fallback court image (recommended size: 800x600px)

## Additional Images (Optional):

- **hero-bg.jpg** - Hero section background image
- **tennis.jpg** - Tennis court image (if tennis courts are added)
- **basketball.jpg** - Basketball court image (if basketball courts are added)

## Image Guidelines:

- Use high-quality images (preferably professional photography)
- Optimize images for web (compress to reduce file size)
- Maintain consistent aspect ratios for court images (4:3 recommended)
- Ensure images are properly lit and showcase the courts clearly
